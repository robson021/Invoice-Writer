Spring Framework + AngularJS. <br />

-html5 <br />
-AngularJS <br />
<br />
-Spring Framework (Spring MVC, Spring Data, Spring AOP (AspectJ), Spring Security) <br />
-H2 embedded database <br />
-Hibernate <br />
-JUnit <br />
-Maven <br />
-iText PDF library
